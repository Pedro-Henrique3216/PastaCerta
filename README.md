# Sistema de Organização de Arquivos Desktop

Este projeto consiste no desenvolvimento de um sistema desktop para organização automática de arquivos, cujo objetivo é classificar e reorganizar arquivos de um diretório com base em suas extensões. O sistema visa melhorar a organização do ambiente digital, reduzir o tempo de busca por arquivos e aplicar boas práticas de desenvolvimento utilizando a linguagem Java.

O objetivo principal deste projeto é automatizar o processo de organização de arquivos, movendo-os para pastas específicas de acordo com suas extensões (por exemplo, arquivos .pdf para a pasta pdfs, arquivos .jpg para jpgs, etc.).

Como objetivos específicos, destacam-se:

Aplicar conceitos de manipulação de arquivos em Java

Utilizar a API java.nio.file

Desenvolver código portátil e reutilizável

Seguir boas práticas de organização e legibilidade de código
